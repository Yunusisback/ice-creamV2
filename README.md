# Ice Cream Shop V2 (TDD Showcase)

An advanced React application demonstrating **Test-Driven Development (TDD)** practices. This project features a polished UI and extensive unit/integration tests to ensure reliability in a shopping cart scenario.


## Key Features

* **TDD Methodology:** Built test-first using React Testing Library and Jest.
* **Integration Testing:** Simulates full user workflows (selecting scoops, checking out).
* **Mock Backend:** Uses JSON Server to simulate REST API responses.
* **Modern UI:** Enhanced design using Bootstrap 5.
* **Async Handling:** Proper error and loading state management.

## Tech Stack

| Category | Technologies |
| :--- | :--- |
| **Core** | React 19 (CRA) |
| **Testing** | React Testing Library, Jest |
| **Styling** | Bootstrap 5 |
| **State** | Context API |
| **Mocking** | JSON Server, Axios |
